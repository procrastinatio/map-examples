Parsing WMTS GetCapabilities with OpenLayers

Output


        

            
With OpenLayers, it is easy to configure a layer parsing an OGC WMTS GetCapabilities document.
            Even layers with DIMENSIONS are easily configured, basically down to (using default):

            
            
            var layer = format.createLayer(capabilities, {
                layer: "ch.bafu.bundesinventare-bln",
                opacity: 0.7,
                isBaseLayer: false
            });
            

            
The topograhic map is a WMS layer and the layer with the polygons is the WMTS parsed layer one.


            
View the ol-wmts-cap.js source to see how this is done.

            
Check the official documentation for more details.

            
Do not forget to add the pro4js library and add the definition for EPSG:21781 (LV03)!

            
            
N.B. You have to register to use the WMTS service on your website.
